My Projects

Explore my hands-on projects in Network Engineering and Cybersecurity, built for real and fictional organizations and companies in Kenya and around the world. My projects demonstrate hands-on expertise in designing secure, scalable network infrastructures and implementing cybersecurity solutions. They cover enterprise network design, firewall configuration, device hardening, wireless deployment and vulnerability assessment. Each project reflects real-world scenarios, showcasing my ability to solve complex challenges and deliver professional, standards-based networking and security solutions.

Enterprise Network Design for Lakeview Level 5 Hospital

I designed a Layer 2 and Layer 3 network using a three-tier architecture for Lakeview Hospital. Departments such as Records, Pharmacy and Labs were assigned unique VLANs. I implemented inter-VLAN routing, DHCP, ACLs for security and static routing for internal communication.

Dual ISP WAN Redundancy for Kisumu County ICT Hub

Built a redundant WAN solution with static routing and floating static routes. ISP1 (Safaricom) was primary and ISP2 (Zuku) served as backup. NAT and route tracking ensured zero downtime. ACLs were added to restrict traffic.

View Project

Campus Network with Layer 3 Switching for University of Wollongong in Dubai

Designed a full campus network using Layer 3 switches at the distribution layer. Configured EIGRP for routing, HSRP for default gateway redundancy and PortFast with BPDU Guard for loop prevention.

View Project

VoIP Network Deployment for Jomo Kenyatta International Airport

Deployed Cisco Call Manager Express on ISR router. Created voice VLANs, configured DHCP for phones and registered multiple IP phones. QoS was implemented to prioritize voice traffic across switches.

View Project

Wireless Network for Nairobi Smart Mall

Configured multiple SSIDs for staff and guests. Integrated WPA2-Enterprise authentication with Radius. Implemented client isolation for guest network and ensured roaming between APs using a controller-based system.

View Project

Basic SD-WAN Simulation for Absa Bank Kenya Branches

Simulated an SD-WAN between Mumbai HQ in India, New York in USA, Kisumu in Kenya and China branches. Used vManage to configure application-aware routing and centralized policy control. Enabled DIA (direct internet access) for branch internet breakout.

Cisco Device Hardening for Lakeview Level 5 Hospital

Implemented SSH, disabled unused services, encrypted passwords, configured banners, set port security on switches and applied AAA with Radius. Conducted password policy enforcement and SNMPv3 setup to ensure industry standards.

Performed internal and external scans on DigiKenya's test network. Identified open ports, outdated Apache versions and weak SSH ciphers. Delivered a professional report including CVSS scores and mitigation strategies and prioritized vulnerabilities using OpenVAS.
